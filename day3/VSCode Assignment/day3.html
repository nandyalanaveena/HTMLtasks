VISUAL STUDIO CODE (VS Code)

What is Visual Studio Code?

Visual Studio Code (famously known as VS Code) is a free open source text editor by Microsoft. VS Code is available for Windows, Linux, and macOS. Although the editor is relatively lightweight, it inculdes some powerful features that have made VS Code one of the most popular development environment tools in recent times. It supports multiple programming languages and is highly customizable through extensions.

Why we use Visual Studio Code?

Visual Studio Code is a widely used and versatile source code editor that offers numerous features, making it a popular choice for developers. Here are the reasons why developers use VS Code:

1. Lightweight and Fast

  • VS Code is a lightweight editor that starts quickly and performs efficiently, even with large projects.

2. Cross-Platform

  • It works seamlessly on Windows, macOS, and Linux, ensuring consistency across different operating systems.

3. Extensive Language Support

  • Out-of-the-box support for popular languages like JavaScript, Python, Java, C++, and many more.
  • Extensions are available to support additional programming languages.

4. Integrated Git Support

  • Built-in Git tools make it easy to manage version control without leaving the editor.
  • Visualize changes, commits, branches, and resolve merge coflicts within the editor.

5. Snippets and Intellisense

  • Auto-complete suggestions, syntax highlighting, and intelligent code navigation.
  • Snippets save time by allowing reusable code templates.

Features of VS Code:

The VS Code user interface allows for a lot of interaction compared to other text editors. To simplify user experience, VS Code is divided into five main regions:

  • The activity bar
  • The side bar
  • Editor groups
  • The panel
  • The status bar

Extensions of VS Code:

Here's a table with popular VS Code extensions and their purposes:

S.No Extension Name Purpose
1. Live Server Launches a local development server with live reload for static files.
2. Live Preview Provides an in-editor browser preview for your application, allowing you to see changes live without switching windows.
3. Prettier Automatically formats your code based on predefined rules, ensuring a consistent and clean code style across your project.
4. Emmet Live Enhances Emmet functionality with real-time code expansion and snippets for faster HTML and CSS coding.
5. Material Icon Theme Adds visually appealing icons to files and folders in the explorer.
6. ESLint Identifies and fix issues in JavaScript and TypeScript code.
7. Docker Adds tools for working with Docker containers and images.
8. Debugger for Chrome Enables debugging of JavaScript code in the Chrome browser.
9. SQLite Lets you explore and manage SQLite databases directly from VS Code.
10. GitLens Enhances Git capabilities with blame, history, and code insights.

Emmets of VS Code:

Here's a table of some commonly used Emmet symbols in VS Code and their purposes:

S.No Emmet Symbol Purpose
1. > Child element creation
2. + Sibling element creation
3. ^ Move up to the parent element
4. * Generate multiple elements
5. {} Add custom text inside an element
6. [] Add attributes to an element
7. # Add an id attribute
8. . Add a class attribute
9. : Add pseudo-classes or pseudo-elements
10. ! Generate a boilerplate HTML document
